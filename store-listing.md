## 상세 설명

### 한국어 (Chrome Web Store 본문)

GitHub Copilot을 사용하다 보면 이런 생각이 드신 적 있으신가요?
"이번 달 Premium Request를 얼마나 썼지?"

Copilot Usage Tracker는 GitHub Copilot의 Premium Request 사용량을 Chrome 툴바에서 바로 확인할 수 있게 해주는 확장 프로그램입니다.

🔢 주요 기능

• 실시간 사용량 배지 — 툴바 아이콘에 현재 사용률(%)이 표시됩니다. 별도로 팝업을 열지 않아도 한눈에 확인 가능합니다.

• 모델별 사용량 분석 — GPT-4o, Claude Sonnet, Gemini 등 어떤 모델을 얼마나 사용했는지 도넛 차트로 시각화합니다.

• 7일 추정치 — 이번 달 일평균 사용량을 기반으로 7일 뒤 예상 사용량을 계산합니다.

• 한도 알림 — 사용량이 80%, 90%, 100%에 도달하면 데스크탑 알림을 발송합니다.

• 자동 갱신 — 백그라운드에서 1시간마다 자동으로 최신 사용량을 갱신합니다.

• 공지사항 — 업데이트 내역, 사용 방법, 알려진 이슈를 확장 내에서 확인할 수 있습니다.

🔒 개인정보 보호

• GitHub Fine-grained Token은 사용자 기기 로컬에만 저장됩니다.
• 외부 서버로 데이터를 전송하지 않습니다.
• 광고, 분석, 트래킹을 사용하지 않습니다.

⚙️ 사용 방법

1. 확장 설치 후 툴바 아이콘 클릭
2. 설정 탭에서 GitHub 사용자명 입력
3. GitHub Fine-grained Token 입력 (Plan: Read-only 권한 필요)
4. 저장 후 사용량 확인

📋 필요 권한 안내

• storage: 토큰 및 사용량 데이터 로컬 저장
• notifications: 한도 도달 시 알림 발송
• alarms: 1시간 주기 자동 갱신
• tabs: 가이드/공지사항 페이지 새 탭으로 열기
• api.github.com: Copilot 사용량 API 호출
• gist.githubusercontent.com: 공지사항 데이터 로드

### English (Chrome Web Store Body)

Ever wondered "How much of my Copilot budget have I used this month?"

Copilot Usage Tracker shows your GitHub Copilot Premium Request usage directly in the Chrome toolbar — no need to dig through GitHub settings.

🔢 Key Features

• Live usage badge — See your current usage % on the toolbar icon at a glance, even without opening the popup.

• Model breakdown — Visualize how much you've used GPT-4o, Claude Sonnet, Gemini, and other models with a donut chart.

• 7-day estimate — Projects your usage for the next 7 days based on your daily average.

• Limit alerts — Get desktop notifications at 80%, 90%, and 100% of your monthly allowance.

• Auto-refresh — Updates your usage in the background every hour.

• Notices page — View changelogs, usage guides, and known issues directly within the extension.

🔒 Privacy First

• Your GitHub Token is stored locally on your device only.
• No data is sent to any external server.
• No ads, no analytics, no tracking.

⚙️ How to Use

1. Click the toolbar icon after installation
2. Enter your GitHub username in the Settings tab
3. Enter your GitHub Fine-grained Token (Plan: Read-only permission required)
4. Save and view your usage

📋 Permission Justification

• storage: Save token and usage data locally
• notifications: Send alerts when usage limits are reached
• alarms: Schedule hourly background refresh
• tabs: Open guide and notices pages in new tabs
• api.github.com: Fetch Copilot usage data
• gist.githubusercontent.com: Load notices/changelog data
