01

수집하는 정보

Copilot Usage Tracker는 서비스 제공에 필요한 최소한의 정보만 수집합니다.

항목 수집 목적 저장 위치 필수 여부
GitHub 사용자명 GitHub API 호출 시 사용자 식별 로컬 (chrome.storage.local) 필수
GitHub Fine-grained Token GitHub API 인증 로컬 (chrome.storage.local) 필수
Copilot 사용량 데이터 사용량 표시 및 캐싱 로컬 (chrome.storage.local) 자동 수집
플랜 설정 (Pro/Business 등) 한도 계산 로컬 (chrome.storage.local) 선택
❌ 수집하지 않는 정보: 이름, 이메일, 결제 정보, 위치 정보, 브라우저 히스토리, 쿠키, 기기 식별자
02

정보의 저장 방식

모든 데이터는 사용자의 로컬 기기에만 저장됩니다. Chrome 브라우저의 chrome.storage.local API를 사용하며, 이 데이터는 확장 프로그램만 접근할 수 있습니다.

  • ✓ 데이터는 사용자 기기 내 Chrome 프로필에만 저장됩니다.
  • ✓ 개발자 서버나 외부 데이터베이스로 전송되지 않습니다.
  • ✓ GitHub Token은 평문으로 저장되므로 기기 보안 관리에 유의하세요.
  • ✗ Cloud 동기화(chrome.storage.sync)는 사용하지 않습니다.
03

정보의 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다.

  • ✓ GitHub API 인증 및 Copilot Premium Request 사용량 조회
  • ✓ 사용량 진행 바, 도넛 차트, 7일 추정치 계산 및 표시
  • ✓ 한도 도달 시 (80%/90%/100%) 데스크탑 알림 발송
  • ✓ 마지막 조회 결과 캐싱 (팝업 빠른 로드용)
04

제3자 제공 및 외부 전송

확장이 통신하는 외부 서비스는 다음과 같습니다.

서비스목적전송 데이터개인정보처리방침
api.github.com Copilot 사용량 조회 GitHub Token, 사용자명 GitHub Privacy
gist.githubusercontent.com 공지사항 JSON 로드 없음 (인증 불필요) GitHub Privacy
✅ 개발자 운영 서버로의 데이터 전송은 없습니다. 광고, 분석, 트래킹 서비스를 사용하지 않습니다.
05

정보의 보관 및 삭제

데이터는 사용자가 직접 삭제하거나 확장을 제거할 때까지 로컬에 보관됩니다.

  • ✓ 확장 팝업 → 설정 탭 → "설정 초기화" 버튼으로 즉시 삭제 가능
  • ✓ Chrome에서 확장 제거 시 모든 로컬 데이터 자동 삭제
  • ✓ 별도의 보관 기간 없이 사용자 요청 즉시 삭제
06

사용자 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • ✓ 열람권: 저장된 데이터는 chrome.storage.local에서 직접 확인 가능
  • ✓ 삭제권: 설정 초기화 버튼 또는 확장 제거로 즉시 삭제
  • ✓ 거부권: 토큰 미입력 시 API 호출 없이 확장 사용 불가 (기능 특성상)
07

보안

GitHub Token은 chrome.storage.local에 저장되며, Chrome 프로필 암호화에 의해 보호됩니다. 단, 기기 분실이나 악성 소프트웨어로 인한 접근은 방어할 수 없으므로 GitHub 계정에서 토큰의 유효 기간을 짧게 설정하고 불필요 시 즉시 폐기하시기를 권장합니다.

💡 보안 권장사항: GitHub Token 발급 시 만료일을 90일 이하로 설정하고, 최소 권한(Plan: Read-only)만 부여하세요.
08

방침 변경

본 방침이 변경될 경우 확장의 공지사항 페이지(버전 업데이트 내역)를 통해 안내드립니다. 중요한 변경이 있을 경우 확장 업데이트 시 팝업 알림을 제공할 수 있습니다.
